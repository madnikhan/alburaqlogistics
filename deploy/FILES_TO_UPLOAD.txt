================================================================================
  UPLOAD METHODS
================================================================================

METHOD 1: FTP/SFTP (FileZilla, WinSCP, etc.)
---------------------------------------------
1. Connect to your Fasthosts FTP server
2. Navigate to your domain root (public_html or www)
3. Upload all files and folders listed above

METHOD 2: SSH/SCP
------------------
scp -r * username@your-server:/path/to/your/domain/

METHOD 3: cPanel File Manager
-------------------------------
1. Login to cPanel
2. Open File Manager
3. Navigate to public_html
4. Upload files via web interface

================================================================================
  AFTER UPLOAD - SERVER SETUP
================================================================================

1. SSH into your server
2. cd /path/to/your/website
3. npm install --production
4. Create .env.local file with your environment variables
5. npm start (or use PM2)

See UPLOAD_INSTRUCTIONS.txt for detailed steps.
