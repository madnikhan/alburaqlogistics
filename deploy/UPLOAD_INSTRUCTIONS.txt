================================================================================
  STEP-BY-STEP DEPLOYMENT
================================================================================

STEP 1: UPLOAD FILES
---------------------
1. Connect to Fasthosts FTP/SFTP server
2. Navigate to your domain's root directory (usually public_html or www)
3. Upload ALL contents of this folder

STEP 2: SERVER SETUP (SSH into your server)
--------------------------------------------
cd /path/to/your/website

# Install dependencies
npm install --production

# Create environment file
nano .env.local
# (Copy from .env.example and fill in your values)
# Save: Ctrl+X, then Y, then Enter

STEP 3: START APPLICATION
---------------------------
# Option 1: Direct start (for testing)
npm start

# Option 2: PM2 (recommended for production)
npm install -g pm2
pm2 start npm --name "alburaq-logistics" -- start
pm2 save
pm2 startup

STEP 4: CONFIGURE DOMAIN
--------------------------
- Point your domain to this server
- Set up SSL certificate (HTTPS)
- Test: https://yourdomain.co.uk
